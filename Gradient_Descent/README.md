## Gradient Descent
Срок сдачи: 13.12.2025 23:59

### Итоги работы

Реализованы несколько методов градиентного спуска:

1) Классический градиентный спуск
2) Градиентный спуск с расисанием (inverse time decay)

Классический градиентный спуск применен к выпуклой функции (квадратичная функция) и к невыпуклой (функция Розенброка).
Градиентный спуск с расписанием применен к невыпуклой функции (функция Розенброка).

Выводы к каждому пункту задания можно найти в ноутбуке.
